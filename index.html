Lists of resources for accessibility evaluation and development

This is a compilation of useful lists of resources to evaluate and develop accessible web or non-web content, as web pages, mobile applications, desktop applications, documents.

It is possible to collaborate, sending proposals for corrections of the lists. For this you can open an issue on GitHub, or send a message to jose.hilera@uah.es , or via twitter at @web_a11y.

List Description
a11y-evaluation-tools Tools that can be used to evaluate the accessibility of web content and non-web content.
a11y-evaluation-examples Examples of accessibility evaluation cases, and examples of web content and non-web content before and after doing corrections to be accessible.
a11y-evaluation-services Organizations that performs accessibility evaluation projects.
a11y-development-resources Useful resources to develop accessible web and non-web content.
Other lists Other useful lists to evaluate or develop accessible web and non-web content.
